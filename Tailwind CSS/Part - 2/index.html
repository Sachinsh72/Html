Tailwind through PostCSS

Development steps for Tailwind are below:

Step 1: Goto terminal run - npm install -D tailwindcss postcss autoprefixer (node js should be installed) Step 2: run - npm install vite Step 3: Goto to package.json and write - ``` "scripts": { "start": "vite" }, ``` Step 4: run - npx tailwindcss init Step 5: Goto tailwind.config.js and write [path of HTML file or *] in content box
