Selectors : 7-8 types of selectors in css majorly used class. basicly used for selecting the html elements that you want to style and they are a fundamental part of working with CSS. In theory these are the patterns used to select elements to which a set of styles will be applied. catagories - Simple selectors, combinators, attribute selectors, pseudo-class selectors, psedudo-elements selectors

Simple Selectors - Universal (*), element (targeted tag name like - h1), class (selector), id (simple), selector list.

Combinators Selectors - Descendent (under or the child selectors like h4 p ), child(all des. selector not the child selector eg h4 > p ), adjacent sibling (just after section + p), general sibling( section ~ p)
